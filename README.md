# AJLP_citation_builder
Instructions to create an AJLP citation builder using custom instructions and a rulebook. AJLP uses APA for literature and a modified form of OSCOLA for legal authorities.

This setup has been tested to work in Google's NotebookLM and Microsoft Copilot Agent.
Upload the AJLP_Combined_Rulebook.md file as knowledgebase and modify the system instructions according to the custom_instructions file.

An implementation of this in NotebookLM can be found here: https://notebooklm.google.com/notebook/c8ce432b-a854-499f-9ba7-08ae3e163848?authuser=3
